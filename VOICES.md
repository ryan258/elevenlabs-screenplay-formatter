# ElevenLabs Voice Reference Guide

A comprehensive reference of available ElevenLabs voices for your screenplay characters. Copy the Voice ID and paste it into the Character Configuration panel.

## How to Use This Guide

1. Find a voice that matches your character's personality
2. Copy the **Voice ID** (the alphanumeric string)
3. Paste it into the "Voice ID" field for that character in the app
4. Adjust voice settings (stability, similarity boost, style) as needed

## Getting Custom Voices

### Option 1: ElevenLabs Voice Library
1. Visit https://elevenlabs.io/voice-library
2. Browse thousands of community-created voices
3. Filter by language, accent, age, and use case
4. Click "Add to My Voices" to access the Voice ID

### Option 2: Clone Your Own Voice
1. Go to https://elevenlabs.io/voice-lab
2. Click "Add Generative or Cloned Voice"
3. Upload 1-25 audio samples of the target voice
4. ElevenLabs will generate a Voice ID for your custom voice

### Option 3: Professional Voice Design
1. Use ElevenLabs Voice Design feature
2. Describe the voice characteristics you need
3. Generate custom voices based on your description
4. Save and use the generated Voice ID

## Voice Settings Tips

Once you've selected a voice, fine-tune it with these settings:

### Stability (0.0 - 1.0)
- **Low (0.0-0.4)**: More expressive, varied, emotional
- **Medium (0.5-0.6)**: Balanced (recommended starting point)
- **High (0.7-1.0)**: More consistent, stable, predictable

### Similarity Boost (0.0 - 1.0)
- **Low (0.0-0.4)**: More creative interpretation
- **Medium (0.5-0.6)**: Balanced
- **High (0.7-1.0)**: Closer to original voice sample

### Style (0.0 - 1.0)
- **Low (0.0)**: More neutral, less expressive
- **Medium (0.5)**: Natural expression
- **High (1.0)**: Maximum expressiveness and emotion

### Speed (0.25 - 4.0)
- **Slow (0.5-0.8)**: Deliberate, careful speech
- **Normal (0.9-1.1)**: Natural speaking pace
- **Fast (1.2-2.0)**: Quick, energetic delivery

## Troubleshooting Voice Issues

### Voice sounds robotic
- **Lower stability** to 0.3-0.5
- **Increase style** to 0.5-0.8
- Try a different voice that's more naturally expressive

### Voice is too inconsistent
- **Increase stability** to 0.6-0.8
- **Lower style** to 0.2-0.4
- Ensure text is properly formatted

### Voice doesn't match character
- Browse the Voice Library for more options
- Try voices from different age groups
- Consider cloning a voice from reference audio

### Audio cuts off or sounds clipped
- Check that your text isn't too long (max ~5000 chars)
- Reduce speed if set above 2.0
- Try a different output format
